ErrorBoundary

ErrorBoundary

A React class component that implements an Error Boundary. It catches JavaScript errors anywhere in its child component tree, logs those errors, and displays a fallback UI instead of the crashed component tree.

This component is used to gracefully handle runtime errors in parts of the UI, preventing the entire application from crashing.
